# Merge Similar Folders

Скрипт для поиска папок с похожими названиями и объединения их содержимого.

## Возможности

✅ **Поиск самых длинных совпадающих названий**  
✅ **Объединение содержимого 2-4 уровня вложенности**  
✅ **Создание родительских папок с общим названием**  
✅ **Интерактивное подтверждение объединения**  

## Запуск

```bash
chmod +x merge_folders.sh
./merge_folders.sh
```

## Как работает

### 1. Поиск совпадений
- Рекурсивно сканирует все папки в текущей директории
- Находит самые длинные общие части названий (минимум 3 символа)
- Показывает найденные пары папок

### 2. Объединение
- Создает родительскую папку с общим названием
- Перемещает содержимое из обеих папок в родительскую
- Удаляет пустые исходные папки
- Обрабатывает конфликты имен

## Особенности

- **Работает с любой глубиной вложенности**
- **Находит самые длинные совпадения** (не только префиксы)
- **Сохраняет структуру файлов** при перемещении
- **Безопасен** - запрашивает подтверждение

## Безопасность

- Интерактивное подтверждение перед объединением
- Обработка конфликтов имен
- Удаление только пустых папок
- Подробное логирование

## Требования

- macOS/Linux
- Bash
- Стандартные утилиты: find, mv, mkdir
